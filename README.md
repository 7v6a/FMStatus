# FMStatus
A very plain and lightweight script that displays information about chosen LastFM Account. Requires an personal API-KEY. <br>
Gives you information about:
- Account Username
- Country
- Registered at (Account Creation Date)
- Playcount (total amount of songs played since connecting to LastFM)
- Currently Played Song
- Last Played Song
- Total time spent listening
